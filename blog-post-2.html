The Building Blocks of Web Design

Week 2

This HTML fundamentals lecture provided an introduction to structuring web pages using proper HTML tags and elements. These fundamentals serve as the building blocks for implementing digital concepts as a designer.

The lecture emphasized organization when coding. Thoughtfully structuring files, naming elements, etc. The hands-on examples in Visual Studio Code provided experience setting up doctypes, titles, headings and other components comprising a page.

Understanding how different tags display content is key knowledge. Seeing heading tags in action establishes hierarchy for presenting information. Adding images via src attributes enables visual storytelling.

Mastering HTML basics builds a technical foundation to create simple or sophisticated page layouts. Reinforcing concepts through consistent practice will develop skills in HTML.

This lecture equipped me with foundational HTML skills to start coding and expand abilities over time. The specific examples and structured approach directly built tangible web development knowledge.
